$$k_{\mathrm{Matern}}(x,x') = 2 \left(1 + 2.23607 r + \frac{5}{3} r^2\right) e^{-2.23607 r},$$
$$r = \sqrt{\frac{(x_1 - x_1')^2}{0.543998^2} + \frac{(x_2 - x_2')^2}{0.351651^2}}\,.$$
$$\mathrm{Cov}[y(x),y(x')] = k_{\mathrm{Matern}}(x,x') + 8.63578\,\delta_{x,x'}. $$
